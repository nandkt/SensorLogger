Deskripsi
Sensor Logger merupakan aplikasi Android yang memanfaatkan sensor **Accelerometer** untuk membaca nilai sumbu X, Y, dan Z secara real-time. Data hasil pembacaan sensor dapat disimpan ke dalam database SQLite dan ditampilkan kembali pada halaman riwayat.

Fitur

- 📱 Membaca sensor Accelerometer
- 💾 Menyimpan data ke SQLite
- 📋 Menampilkan riwayat data
- 🗑 Menghapus seluruh data
- 📅 Menyimpan tanggal dan jam
- 🟢 Menampilkan status Stabil atau Bergerak

Teknologi

- Java
- Android Studio
- SQLite Database
- XML Layout
- Accelerometer Sensor

Cara Menjalankan

1. Clone repository ini.
2. Buka menggunakan Android Studio.
3. Jalankan pada perangkat Android yang memiliki sensor Accelerometer.
4. Tekan tombol **Simpan Data** untuk menyimpan hasil pembacaan sensor.
